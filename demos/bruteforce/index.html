Brute Force Visualizer

What is password cracking?

"Password cracking" is when somebody works out what a password is. This demonstration is a visualisation of the most basic way that hackers can "crack" passwords: using computers to try every password they can.

Computers can do this very, very quickly. Well secured websites make this difficult and can slow it down to only thousands of guesses per second, but it can be billions of guesses per second!

How to use the demonstration:

  • Compare character sets: Use the checkboxes (Uppercase, Numbers, Symbols) to create different password complexity levels for Password A and Password B.
  • Select speed: Choose how many password combinations are tried per second. Start small so you can see what's going on, then speed it up to see how quickly hackers can try different combinations!
  • Start the simulation: Click "Start" to begin and watch the guesses happen in realtime.
  • Observe the difference: The dots (•) show password length milestones. Notice how much longer it takes to reach each milestone when uppercase, numbers, or symbols are added. But, get to a certain length and even all lower case can produce very strong passwords.

Key Takeaway: Most hackers don't have to do much to guess passwords - they just run a program and wait for it to crack the simple passwords.
